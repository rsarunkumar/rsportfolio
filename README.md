# Portfolio Website 🌐

📌 How to Use the ZIP File

Download & Extract the ZIP file.

Open the folder.

Double-click index.html → it will open in your browser. ✅

🚀 Features
- Responsive design  
- Smooth scroll-to-top button  
- Social media integration (GitHub, LinkedIn, Instagram)  
- Simple and clean UI
